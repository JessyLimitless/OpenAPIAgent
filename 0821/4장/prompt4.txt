
Streamlit 앱을 개발하고자 합니다. 이 앱은 OpenAI의 GPT-3.5 Turbo 모델을 사용하여 복잡한 과학 주제를 초등학생도 쉽게 이해할 수 있도록 설명해주는 기능을 포함해야 합니다. 

다음 요구 사항을 충족해야 합니다:

1. UI 구성: Streamlit을 사용해 사용자 인터페이스를 만듭니다.
2. GPT 모델과의 상호작용: LangChain 라이브러리를 사용해 GPT-3.5 Turbo 모델과 상호작용합니다.
 3. OpenAI API 키를 하드 코딩하여 설정해 주세요. 여기서 모델의 창의성(temperature)을 0.99로 설정해 주세요. 
4. 사용자 입력: 사용자가 과학 주제를 입력할 수 있는 입력 필드를 제공합니다.
5. 설명 생성: 사용자가 입력한 과학 주제에 대해 GPT 모델이 간단한 문장 구조와 일상적인 예시를 사용해 설명을 생성합니다. 설명은 초등학생이 이해할 수 있는 수준이어야 합니다.
6. 결과 표시: 생성된 설명을 Streamlit 화면에 표시합니다.

이 요구 사항에 따라 Python 코드를 작성해 주세요.

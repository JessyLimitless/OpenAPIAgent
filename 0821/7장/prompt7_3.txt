Streamlit과 OpenAI의 GPT-3.5-turbo 모델을 사용하여 보도자료 생성 애플리케이션을 만드는 Python 코드를 작성해 주세요. 이 애플리케이션은 다음 기능을 포함해야 합니다:

1. 입력 필드 제공: 사용자가 보도자료를 작성하기 위해 다음 정보를 입력할 수 있는 필드를 제공합니다:                    주제, 주요 내용, 발표 날짜, 회사 정보.
2. 보도자료 생성: 사용자가 입력한 정보를 바탕으로 GPT-3.5-turbo 모델을 사용해 설득력 있고 전문적인                   보도자료를 생성합니다.
3. 결과 출력: 생성된 보도자료를 화면에 출력합니다.
4. 오류 처리: 보도자료 생성 중 오류가 발생할 경우, 사용자에게 오류 메시지를 표시합니다.
5. 간단한 인터페이스: Streamlit을 사용해 간단한 UI를 구성하고, 사용자가 입력한 내용을 제출하면 결과를 화면에 표시합니다.

OpenAI API 키는 코드에 직접 하드코딩해 주세요.
